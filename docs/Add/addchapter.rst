======================================================================
addchapter
======================================================================
------------------------------------------------------------
Aliases: ac, addch, addc
------------------------------------------------------------
Description
==============
Add a project to the database.

Required Role
=====================
Role `Akashi's Minions`.

Arguments
===========
Required
---------
:chapter: Project and chapter number of the chapter to add.
:raws: Link to raws on Box.

Optional
------------
:tl, rd, ts, pr: Staff for the chapter.
