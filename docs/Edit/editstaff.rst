editstaff
======================================================================
------------------------------------------------------------
Aliases: 
------------------------------------------------------------
Description
==============
Edit a staffmembers attributes.

Required Role
=====================
Role `Neko Herders`.

Parameters
===========

Required
---------
:member: Staffmember to edit.

Optional
------------
:id: The staff's discord id.
:name: The staff's username.
:status: Can be "active" or "inactive".
