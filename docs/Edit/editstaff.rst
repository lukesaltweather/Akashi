======================================================================
editstaff
======================================================================
------------------------------------------------------------
Aliases: 
------------------------------------------------------------
Description
==============
Edit a staffmembers attributes.

Required Role
=====================
Role `Neko Herders`.

Arguments
===========

Required
---------
:member:
    | Staffmember to edit. [:doc:`/Types/staff`]

Optional
------------
:id:
    | The staff's discord id. [:doc:`/Types/number`]
:name:
    | The staff's username. [:doc:`/Types/text`]
:status:
    | Can be "active" or "inactive". [:doc:`/Types/literals`]
