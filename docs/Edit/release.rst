======================================================================
release
======================================================================
------------------------------------------------------------
Aliases: 
------------------------------------------------------------
Description
==============
Mark a chapter as released.

Required Role
=====================
Role `Neko Workers`.

Arguments
===========
Required
---------
:chapter:
    | The chapter to set as released, in format: projectName chapterNbr [:doc:`/Types/chapter`]

Optional
------------
:date: The date on which it was finished, defaults to current date and time.
    | [:doc:`/Types/datetime`]
