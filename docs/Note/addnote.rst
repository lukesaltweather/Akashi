======================================================================
addnote
======================================================================
------------------------------------------------------------
Aliases: an, note, annotate
------------------------------------------------------------
Description
==============
Add a note to the specified chapter.

Required Role
=====================
Role `Neko Workers`.

Arguments
===========

Required
------------
:chapter:
    | The chapter to add the note to. [:doc:`/Types/chapter`]
:text:
    | The actual text of the note to add. [:doc:`/Types/text`]
