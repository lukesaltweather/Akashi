======================================================================
addnote
======================================================================
------------------------------------------------------------
Aliases: an, note, annotate
------------------------------------------------------------
Description
==============
Add a note to the specified chapter.

Required Role
=====================
Role `Neko Workers`.

Parameters
===========

Required
------------
chapter: :doc:`chapter`
    The chapter to add the note to.
text: :doc:`text`
    Thw actual text of the note to add.
