# $addnote

## Description

Appends a note to the existing notes in the chapter. You cannot edit nor delete a note, unless you are an admin.

## Permissions

Requires role `Neko Worker`.

## Usage

### Required

`-p=` Project title or altName

`-c=` Chapter number

`-note=` Note to append
